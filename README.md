# openpecha-forum-posts

Research write-ups published to the OpenPecha community forum, covering data cleaning, topic
modeling, and knowledge graph experiments conducted in support of Tibetan-to-English machine
translation work.

- `DataCleaning/` — heuristics and processes for cleaning translation training data
- `TopicModeling/` — clustering and domain-tagging experiments for the translation dataset
- `KnowledgeGraphs/` — knowledge graphs for accessible machine-generated translations and commentaries
- `GyalpoTowardsBuddhistQnA.md` / `.pdf` — toward a Buddhist Q&A system
- `TechReportDescription.md` — a style guide for writing technical reports, used across these posts

Originally part of the [MLotsawa](https://github.com/billingsmoore/MLotsawa) repo, split out into its
own repo since it's a body of published writing rather than active translation-project code.
